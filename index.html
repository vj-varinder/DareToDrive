Instructions for playing

This game has three levels

  1. Beginner
    • This level would have traffic.
    • User can tap on left part to move left.
    • User can tap on right part to move right.
  2. Moderate
    • This level would have traffic and coins.
  3. Expert
    • This level would have traffic, coins and spikes.
